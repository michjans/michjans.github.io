    Daar kunnen eventueel ook oudere versies gedownload worden. Of gebruik onderstaande directe links, afhankelijk van uw OS.

    Windows

    De applicatie is getest op Windows 10, maar zou ook moeten werken op Windows 11. Een installer kan gedownload worden die Route Generator installeert, inclusief FFmpeg en bmp2avi.

    Deïnstalleer eerst een oudere versie van Route Generator als deze nog geïnstalleerd is.

    v2.0.0 routegen-win64-2.0.0.exe 64 bits
    v1.12.0 routegen-win64-1.12.0.exe 64 bits
    v1.9.1 (laatste 32 bits versie) routegen-win32-1.9.1.exe 32 bits

    Linux

    Ubuntu gebruikers kunnen een installer downloaden. Executable maken (chmod +x) en uitvoeren. De Route_Generator-x86_64.AppImage wordt niet meer ondersteund en is alleen nog beschikbaar voor oudere versies.

    v2.0.1 routegen-linux-ubuntu-24.04-2.0.1.run Ubuntu 24.04 64 bits
    v2.0.0 routegen-linux-ubuntu-22.04-2.0.0.run Ubuntu 22.04 64 bits
    v1.12.0 routegen-linux-ubuntu-22.04-1.12.0.run Ubuntu 22.04 64 bits
    v1.11.0 routegen-linux-ubuntu-20.04-1.11.0.run Ubuntu 20.04 64 bits
    v1.10.0 Route_Generator-1.10.0-x86_64.AppImage AppImage 64 bits
