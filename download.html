    Download

    The latest version is available on the SourceForge project page, where you can optionally download older versions as well. Or use the direct links below, depending on your OS.

    Windows

    The application has only been tested on Windows 10, but should also still work on Windows 7 and 8. An installer can be downloaded, which install Route Generator incuding FFpmeg and bmp2avi. If an older version of Route Generator is still installed, it can automatically uninstall de old version, which is recommended.

    v1.10.0 routegen-win64-1.10.0.exe 64 bits
    v1.9.1 routegen-win32-1.9.1.exe 32 bits

    Linux

    Linux users can download the Route_Generator-x86_64.AppImage. It doesn't need installation. Just download, make executable (chmod +x) and run. It has only been tested on Ubuntu 18, but should also work on similar distributions.

    v1.10.0 Route_Generator-1.10.0-x86_64.AppImage 64 bits
    v1.9.1 Route_Generator-1.9.1-x86_64.AppImage 64 bits

    Source code

    The source code builds on Windows and Linux, but should also build on mac OS. I have not tested this myself.
